# healthcare-platform
Digital platform for mind-body health solutions


## Getting Started

1. Install Node.js
2. Clone the project
3. Install dependencies: `npm install`
4. Start: `npm start`

Mission:
Build digital tools to improve mind-body health.

Goals:
- Meditation apps
- Personalized wellness programs
- Digital community
